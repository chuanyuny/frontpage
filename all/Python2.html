1.HDFS操作样例(Python2)

在Client终端中执行HDFS命令

1. ls命令-显示HDFS文件系统根目录的所有文件
 hdfs dfs -ls / 或者 hadoop fs -ls /

终端输出内容如下，列出所有HDFS文件系统根目录的内容：


drwxrwxrwx - yarn hadoop 0 2017-04-10 01:43 /app-logs
drwxr-xr-x - hdfs hdfs 0 2017-04-09 17:31 /apps
drwxr-xr-x - yarn hadoop 0 2017-04-09 17:28 /ats
drwxr-xr-x - mapred hdfs 0 2017-04-09 17:28 /mapred
drwxrwxrwx - mapred hadoop 0 2017-04-09 17:28 /mr-history
drwxr-xr-x - root root 0 2017-04-14 06:00 /root
drwxrwxrwx - spark hadoop 0 2017-04-18 03:46 /spark-history
drwxrwxrwx - hdfs hdfs 0 2017-04-09 17:32 /tmp
drwxr-xr-x - hdfs hdfs 0 2017-04-10 01:40 /user
2. put命令-将root用户下的文件传到HDFS文件系统中
hdfs dfs -put /root/test.txt  /root

执行hdfs dfs -ls 命令查看操作结果：

hdfs dfs -ls /root/test.txt

终端输出内容如下, 文件已经被上传到HDFS文件系统中：

-rw-r--r-- 3 root root 22 2017-04-19 02:03 /root/test.txt
